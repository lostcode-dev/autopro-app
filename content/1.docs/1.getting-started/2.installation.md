## Requisitos

- Node.js em versão compatível com o projeto
- pnpm

## Instalar dependências

```bash
pnpm install
```

## Rodar em desenvolvimento

```bash
pnpm dev
```

## Verificações recomendadas

```bash
pnpm lint
pnpm typecheck
```

## Estrutura importante para desenvolvimento

- `app/pages/app/**` concentra o fluxo da oficina
- `app/pages/admin/**` concentra o fluxo administrativo
- `server/api/**` concentra as rotas server-side
- `supabase/**` concentra banco, migrações e funções relacionadas

> Regra importante: integrações com banco e serviços externos devem acontecer no server-side, via `server/api/**` ou funções apropriadas.
