O **AutoPro** é um sistema para oficinas mecânicas com foco em **ordens de serviço**, **clientes**, **veículos**, **financeiro**, **compras** e **estrutura administrativa**.

A aplicação está sendo consolidada sobre **Nuxt** no frontend e **Supabase** no backend, com migração por fases para preservar os fluxos mais importantes antes de avançar para módulos mais complexos.

## Como o sistema está organizado

- **`/app`**: fluxo operacional da oficina
- **`/admin`**: fluxo administrativo global
- **`/consulta`**: rota pública e isolada para casos específicos

## O que já foi migrado na base

- banco de dados para o novo schema
- funções essenciais de usuários e organizações
- funções principais de ordens de serviço
- fluxo de billing e Stripe

## O que ainda evolui por fases

- módulos completos de pages no frontend
- relatórios avançados
- parte fiscal/Nuvem Fiscal
- refinamentos de permissões, navegação e experiência por contexto
